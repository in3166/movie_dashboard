<h2 id="installation">
    2. 설치 및 실행 방법
</h2>

### Back-End

~~~javascript
//mongodb 설치(mac을 기준으로 brew로 설치하였습니다. windows 설치 방법과는 다를 수 있음)
$> brew install mongodb

$> cd calendar-back

//install dependency
$> npm install

//start server
$> npm start
~~~

### Front-End

~~~javascript
//angular cli 설치 (Node >= 8.9 필요)
$> npm install -g @angular/cli

$> cd calendar-front

//install dependency
$> npm install

//dev server 실행. `http://localhost:4200/`으로 접속
$> ng serve
~~~
